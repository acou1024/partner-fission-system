# 合伙人分销管理系统

## 一、项目背景与意义

在私域流量运营场景中，企业通过合伙人模式进行客户裂变与分销，但传统的线下管理方式存在**佣金核算复杂、客户归属不清、数据不透明**等痛点。本项目旨在开发一套合伙人分销管理系统，帮助企业实现合伙人管理、客户归属追踪、佣金自动计算与结算的数字化管理，提升运营效率和数据透明度。

## 三、核心功能模块（Backlog）

### Must-have（必须实现）

| 模块 | 功能说明 |
|------|----------|
| 用户认证 | 管理后台账号登录、微信OAuth合伙人登录 |
| 合伙人管理 | 合伙人的增删改查、层级关系维护（支持两级） |
| 客户管理 | 客户信息管理、归属合伙人绑定 |
| 邀请码系统 | 合伙人专属邀请码/链接，客户扫码自动归属 |
| 流水录入 | 客户交易流水的录入与查询 |
| 佣金计算 | 根据分润规则自动计算合伙人佣金 |
| 结算管理 | 佣金结算记录与状态跟踪 |
| 合伙人工作台 | H5端查看客户、佣金、团队数据 |

### Nice-to-have（可选实现）

| 模块 | 功能说明 |
|------|----------|
| 素材中心 | 营销素材的上传与分发 |
| 话术库 | 分类话术管理，供合伙人使用 |
| 客户报备 | 合伙人报备客户，员工审核 |
| 域名防封 | 多域名池管理，入口与落地域名分离 |
| 数据大盘 | 可视化数据概览与排行榜 |
